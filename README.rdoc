== Accounts on React.js

{Accounts on React.js}[https://accounts-manager-on-react.herokuapp.com/] is a Ruby on Rails application that utilizes React.js in the frontend. It was built as an introduction to the functionality of React.js and served as a basic tutorial into the dynamics and strucutre of React.js. It was also built to better understand the {react_rails gem}[https://github.com/reactjs/react-rails] and how it integrates into Ruby on Rails projects
